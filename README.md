## Instalação

 
1. `cd client`
2. `npm install` ou `yarn install`
3. `npm start` ou `yarn start`
4. `cd server`
5. `npm install` ou `yarn`
6. `npm rum devStart`

Altere seus dados do banco de dados mySQL `server/index.js`
